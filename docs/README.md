# Documentation

This folder contains the extended methodological documentation for the study.

## File

* `Methodology_Extended.docx` – extended methodology document describing the research methodology, data processing, operationalization of variables, statistical analysis strategy, and empirical findings related to SDG involvement among SMEs.

## Content overview

The document includes:

* sample characteristics and descriptive overview;
* operationalization of key SDG involvement variables;
* construction of motivational, institutional, resource-related and value-based variables;
* analytical strategy for RQ1 and RQ2;
* description of statistical methods, including descriptive statistics, ranking procedures, correlation analysis, non-parametric tests, FDR correction and regression models;
* extended presentation of the empirical findings.

This document is intended to provide a more detailed explanation of the methodology and analytical workflow behind the scripts, tables and figures included in the repository.
