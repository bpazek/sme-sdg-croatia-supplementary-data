# Scripts

This folder contains the Python scripts used for the empirical analysis of the study on SDG involvement among SMEs.

The scripts are organized into separate subfolders according to the analytical part of the study. Each subfolder contains the scripts for one analysis block and, where relevant, its own README file with more detailed information.

## Folder structure

* `DescriptiveStatistics/` – contains the script used for the descriptive overview of the sample and the initial descriptive statistics.
* `RQ1/` – contains the scripts used for Research Question 1, which examines which SDGs dominate the operations and sustainability efforts of Croatian SMEs.
* `RQ2/` – contains the scripts used for Research Question 2, which examines motivational, institutional, resource-related and value-based factors associated with SDG involvement.
* `SupplementaryCorrelation/` – contains the scripts used for the supplementary correlation analysis between external funding sources and motives for implemented improvements.

## Analysis workflow

The scripts are intended to be run in the order defined within each subfolder. In general, the workflow follows these steps:

1. validate the relevant coded survey variables;
2. construct derived analytical variables and indices;
3. produce descriptive summaries and rankings;
4. estimate statistical associations or models;
5. generate figures and final summary outputs.

The scripts were developed and run in Google Colab using the coded survey dataset. Input and output file paths are written accordingly. If the scripts are run in another environment, the file paths may need to be adjusted.

## Notes

The folder separates the analysis into transparent and reproducible stages. This makes it easier to inspect how each table, figure and statistical result was produced. The output files generated by these scripts are stored separately from the scripts themselves.
