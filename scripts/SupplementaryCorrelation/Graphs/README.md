# Graphs

This folder contains the figures generated for the supplementary analysis of the relationship between external funding sources and motives for implemented improvements.

The analysis connects q34 variables, which describe external funding sources used during the previous three years, with q31 variables, which describe motives for implemented improvements.

## Main figures

* `AA3_Figure_1_main_bubble_funding_vs_motivators.png` – bubble plot showing the relationship between the number of external funding sources used and the number of positive improvement motives reported. Bubble size and the value inside each bubble represent the number of firms in each observed combination.
* `AA3_Figure_2_mean_motivators_by_funding_category.png` – comparison of the mean number of positive improvement motives between firms that reported selected funding categories and firms that did not report them.

## Appendix figures

* `AA3_Appendix_Figure_A1_phi_heatmap_q34_q31.png` – exploratory heatmap of phi coefficients for individual external funding source × improvement motive pairs.
* `AA3_Appendix_Figure_A2_top_exploratory_phi_pairs.png` – bar chart showing the largest exploratory individual funding–motive associations according to the absolute phi coefficient.

The main figures are put in the main text of our article. The appendix figures provide additional exploratory documentation, because no individual funding–motive pair remained statistically significant after FDR correction.
